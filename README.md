# Nuestra Historia

Página web romántica para celebrar nuestra relación.

## Despliegue en GitHub Pages

1. Sube todos los archivos a un repositorio de GitHub.
2. Ve a Settings > Pages.
3. Selecciona la rama main y la carpeta raíz.
4. Guarda y espera a que se despliegue.

## Estructura

- `index.html`: Página de inicio
- `historia.html`: Nuestra historia
- `fotos.html`: Galería de fotos
- `lugares.html`: Lugares que queremos visitar
- `restaurantes.html`: Restaurantes visitados
- `gatas.html`: Nuestras gatas Fifi y Nucita
- `nufi.html`: Nuestro emprendimiento NUFI

## Personalización

- Reemplaza las imágenes en `assets/` con tus fotos reales.
- Edita los textos en los archivos HTML.
- Ajusta colores en `styles/main.css`.
